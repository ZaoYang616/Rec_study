# Sharedbottom_PEPNet_demo

本项目是一个面向电商推荐场景的多级联合预估模型开源示例。核心模型实现了针对 **点击率 (CTR)** 和 **转化率 (CVR)** 的多任务学习框架，融合了 `ShareBottom` 底层共享网络、`PEPNet`（参数个性化网络）、`DIN`（深度兴趣网络）以及 `AFM`（注意力因子分解机）。该框架有效解决了推荐系统的多任务负迁移、数据极度不平衡及难以捕获深度交叉特征等痛点。

---

## 1. 数据来源

本项目针对阿里提供的公开推荐数据集：**Ali-CCP (Alibaba Click and Conversion Prediction)**。  
这是包含淘宝推荐系统实际展示、点击与购买日志的大型数据集，包含亿级别的曝光数据，涵盖极大规模的稀疏特征、用户序列以及多目标的转化体系。

- **获取地址**: 您可以在阿里天池官方或各类开源推荐算法仓库通过授权下载。
- **预处理**: 将解压的源样本文件和特征文件按照预定路径摆放，运行自带的 `opendata_V2_success.ipynb`，即可自动化进行数据连接洗牌及构建面向 Embedding 的序列稠密版 Parquet 数据结构，并自动打包了特征维度的全局词表，剔除了异常样本。

---

## 2. 核心架构与模型描述

系统基于 PyTorch 编写，集成封装于 `modelV5.py`。核心流水线如下：

1. **Embedding 查询与拼接 (MultiHotEmbeddingSum)**：将无边界的字符串 ID 转化为统一向量空间，采取 Sum Pooling 或供进阶 Attention 提取，辅以 `LayerNorm` 的数值稳定机制保护。
2. **场景增强门控网络 (EPNet)**：基于用户当前场景特征去生成 Gate 参数控制过滤物品特征，生成“特定环境下的目标物料”。
3. **目标兴趣注意力 (DINAttentionLayer)**：引入阿里核心逻辑的深度行为提纯算法。用前序算出的特定目标物料特征当查询种子，向用户极其长的历史记录行为提纯有针对性的注意力分布网络并动态集成最终画像。
4. **注意力交叉计算 (AFM Layer)**：穷举所有的领域特征(Field)计算它互相之间的二元内积特征并再通过内置的微型注意力网络求和成特征组合因子。
5. **CTR底层打分塔 (Shared Bottom)**：把前面四大产出的向量组进行拼接，流入 MLP 网络进行最原汁原味的 CTR 训练估算分发，形成后续转化动作的重要桥底隐藏特征。
6. **转化过滤分离门控与 PEPNet (CVR 打分塔)**：通过断开 CTR 塔带来的负梯度迁移干扰(`.detach()`)，组装上专有的个人画像生成网络控制转化网络的每一层参数系数分配。最后将提纯的深层非对称特征打出 CVR 成绩。

## 3. 环境与参数配置

### 环境依赖
本项目提供专门记录测试通过的运行环境变量文本（如搭配 RTX5090 等高端发烧级硬件的高性能编译）。
- Python 3.9+
- OS: Windows/Linux
- `torch`, `pandas`, `pyarrow`, `scikit-learn`, `matplotlib`

详细请参考 `environment_setup_RTX5090.txt` 里的匹配版本项或可以直接通过 `pip install -r` 挂载对应依赖。

### 核心参数超参
网络训练的调参设置可以在主流程代码中找到。默认推荐如下：
- `embed_dim = 32` : 表示每个稀疏类别的内隐向量长度
- `BATCH_SIZE = 2048` : 数据吞吐量大可以拉高，需视显存容量浮动
- `EPOCHS = 5` : 结合内建的早停保护机制防止 CVR 过拟合
- `LEARNING_RATE = 5e-4` : 配合 AdamW 修正训练
- `BOTTOM_DROPOUT = 0 / CVR_DROPOUT = 0.05` : 对应由于高稀疏目标避免冲断微弱梯度的分塔设计
- **Focal Loss** `alpha / gamma`：已预先配置了解决 CVR 高达上万比一的正负极大失衡衰减比例。

## 4. 如何运行

### a) 准备并拉取数据
将 `Ali-CCP` 数据安放好后，打开并运行预处理笔记本组件：
```bash
jupyter notebook opendata_V2_success.ipynb
```
依次执行 Cell 解析数据集后将产出 `.parquet` 和词表大小索引 `.json` 等流出给模型的专用清洗格式文件。

### b) 启动模型组装评估与端到端训练
接着切换前往：
```bash
jupyter notebook mainV6_result.ipynb
```
确认修改代码里 `train_parquet` 及相关引用路径为您电脑的生产路径后，依次点击运行 `mainV6_result.ipynb` 中单元格即可。笔记本内置了可视化呈现损失值与评估预测精度的动态图表展现以及提前早停结束功能逻辑闭环。
